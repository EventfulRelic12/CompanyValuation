Company Discounted Cashflow Valuation

Note: all numberic values default to 0.

How It Works

To calculate the Company's value based on the present value of future cashflows, enter the following information into the respective text-boxes:
1. Stock Ticker - a series of capital letters to represent a valid company's stock ticker. (Example: APPL)
2. Growth-Rate - a number value representing the percent growth the company expects to have in their cashflow. (Example: 10 or 10.5)
3. Terminal Price Multiple - a number value representing how many times greater the selling price is than the company's value. (Example: 10 or 10.5)
4. Desired Return - a number value representing the percent return desired from the company each year. (Example: 10 or 10.5)
5. Number of Years - a number value representing the number of years the company is allowd to grow before selling. (Example: 10 or 10.5)

NOTE: Due to the API being used, there is a limited number of times companies can be evaluated.
Additionally, if the input values of growth-rate, terminal price, desired return, or number of years are not a number
they will default to 0.
